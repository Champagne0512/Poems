- **快速导航**: 卡片式导航入口

## 技术架构

### 前端技术栈
- **框架**: Vue 3 + Composition API
- **状态管理**: Pinia
- **路由**: Vue Router
- **样式**: CSS3 + CSS变量系统
- **构建工具**: Vite

### 后端技术栈
- **数据库**: Supabase (PostgreSQL)
- **认证**: Supabase Auth
- **API**: RESTful API + 存储过程

## 数据库设计

### 核心表结构
1. **poems** - 诗词主表
2. **profiles** - 用户资料表
3. **favorites** - 收藏关系表
4. **comments** - 评论表
5. **browse_history** - 浏览历史表

### 存储过程
- `search_poems` - 全文搜索
- `get_popular_poems` - 热门诗词
- `get_recent_poems` - 最新诗词
- `increment_poem_view_count` - 增加浏览次数
